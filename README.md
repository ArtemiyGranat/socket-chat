# Socket Messenger
## Requirements
* Google Chrome/Chromium
* Python 3.8 and newer
* Eel library
* Pycryptodome library 

## Setup
```zsh
pip install -r requirements.txt
```

## Usage
### Server side:
```zsh
python server.py
```
### Client side:
```zsh
python client.py <server-ip>
```

## Current features
- [x] Client-server connection
- [x] Text messaging
- [x] Encrypting messages with AES encryption
- [x] GUI
- [x] Private messages
- [X] File transfer
