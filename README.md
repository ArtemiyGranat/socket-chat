# LAN Messenger
## Requirements
* Python 3.8 and newer
* Eel library
* Pycryptodome library 

## Setup
```zsh
pip install -r requirements.txt
```

## Usage
### Server side:
```zsh
python server.py
```
### Client side:
```zsh
python client.py <server-ip>
```
