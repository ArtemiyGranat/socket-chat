# LAN Messenger
## Requirements
* Python 3.8 and newer
* Eel library
* Pycryptodome library 
```zsh
pip install eel
```

```zsh
pip install pycryptodome
```

## Usage
### Server side:
```zsh
python server.py
```
### Client side:
```zsh
python client.py <server-ip>
```
